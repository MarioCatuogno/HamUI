# Shaman Macros
Macros for Shaman specializations.

## Shared
```
#showtooltip Healing Surge
/cast [@mouseover, exists, noharm, nodead][@target, exists, noharm, nodead][] Healing Surge
```
```
#showtooltip Lightning Bolt
/startattack
/cast Lightning Bolt
```
