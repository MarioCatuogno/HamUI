# Monk Macros
Macros for Monk specializations.

## Shared
```
#showtooltip Effuse
/cast [@mouseover, exists, noharm, nodead][@target, exists, noharm, nodead][] Effuse
```
```
#showtooltip Tiger Palm
/startattack
/cast Tiger Palm
```
